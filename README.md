🌟 Proje Hakkında
Velocity, modern iş dünyasının hızına ayak uydurmak isteyen yazılımcılar için tasarlanmış bir kariyer asistanıdır. Sadece bir yol gösterici değil, aynı zamanda mülakat stresini üzerinizden atmanızı sağlayan bir antrenördür.

🤖 AI Mülakat Simülatörü: Teknik yeteneklerinize göre özelleştirilmiş, gerçek zamanlı geri bildirim veren mülakatlar.

🛣️ Akıllı Yol Haritaları: Mevcut seviyenizi analiz ederek gitmek istediğiniz pozisyona en kısa rotayı çizer.

📊 Analitik Raporlama: Gelişim sürecinizi verilerle takip edin.

🔗 Canlı Demo Linki

https://velocity-ai-mentor.vercel.app/


Not: Demo sunucusu "cold start" (uyku modu) modunda olabilir. İlk yanıtın gelmesi yaklaşık 45-60 saniye sürebilir. Sabrınız için teşekkürler! ☕


## 🛠️ Teknolojiler

**Frontend:**
- React.js (Vite)
- Tailwind CSS
- Lucide React Icons
- Canvas Confetti
- Axios

**Backend:**
- Node.js & Express.js
- Google Generative AI (Gemini API)
- Cors & Dotenv
